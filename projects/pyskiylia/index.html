What is Skiylia?

Skiylia is a custom programing language I started developing back in March of 2021. The language is dynamically typed, object oriented, and most importantly *interpreted*, with a strong focus on code readability and understandability. While it may share many similarities with C derivatives, its heritage is definitely Pythonic.

The first version of skiylia began with a relatively slow Abstract Syntax Tree interpreter built in pure python. This provided an excellent real-world test bed for language features, as everything had been purely theoretical before that point. This version of Skiylia, PySkiylia, is currently version , and represents the first itteration of the language.
